# 🚢 Морський бій 2026 (Battleship 2026)

## 📌 Опис проєкту

Морський бій 2026 — це сучасна реалізація класичної гри "Морський бій", написана на Python з використанням бібліотеки pygame.

Гра має:

* красивий графічний інтерфейс
* анімації вибухів
* синтезовані звуки (без використання готових аудіофайлів)
* режим гри проти ШІ або іншого гравця
* обмеження часу на хід

---

### ▶️ Запуск

python main.py

---
## 🎮 Основні можливості

### 🧠 Режими гри

* Проти ШІ (AI)
  Граєш проти комп'ютера з простою логікою пошуку кораблів

* PvP (2 гравці)
  Два гравці грають по черзі на одному комп'ютері

---

### ⚙️ Функціонал гри

#### 🚢 Розстановка кораблів

* Ручна або автоматична (A або кнопка "АВТО")
* Обертання корабля — клавіша Пробіл
* Перевірка правильності розміщення (без накладення і дотику)

#### 💥 Бойова система

* Постріли по клітинках
* Стани клітин:

  * Порожньо
  * Корабель
  * Попадання
  * Промах
  * Знищений корабель

#### ⏱️ Таймер

* 30 секунд на кожен хід
* Якщо час вийшов — автоматичний пропуск

#### 🤖 Штучний інтелект

* Випадкові постріли
* "Добивання" корабля після попадання (hunt mode)

#### 🔊 Звуки (синтезовані)

* Попадання
* Промах
* Вибух корабля
  (генеруються через numpy і pygame.sndarray)

#### ✨ Візуальні ефекти

* Анімація вибухів (частинки)
* SVG-графіка (кораблі, море, UI)
* Неонові рамки
* Пульсуючий заголовок

---

## 🧱 Структура проєкту

### Основні класи

#### Game

Головний клас гри:

* керує фазами гри
* обробляє події
* малює інтерфейс
* запускає цикл гри

#### Board

Логіка ігрового поля:

* зберігає сітку
* розміщує кораблі
* обробляє постріли
* перевіряє перемогу

#### AI

Штучний інтелект:

* обирає координати
* аналізує попадання
* добиває кораблі

#### PlacementUI

Система розстановки кораблів:

* черга кораблів
* попередній перегляд

#### SoundManager

Система звуку:

* генерує звуки через математичні функції
* не використовує аудіофайли

#### Explosion і Particle

Система анімацій:

* створює ефект вибуху
* частинки рухаються і зникають

## 🎯 Управління

| Дія               | Кнопка       |
| ----------------- | ------------ |
| Обертання корабля | SPACE        |
| Авто-розстановка  | A            |
| Постріл           | ЛКМ (мишкою) |
| Меню              | Мишка        |

---

## 🧩 Ігрові фази

1. MAIN_MENU — головне меню
2. P1_PLACE — розстановка гравця 1
3. P2_PLACE — розстановка гравця 2
4. PLAYER_TURN — хід гравця 1
5. P2_TURN — хід гравця 2
6. AI_TURN — хід ШІ
7. GAME_OVER — кінець гри
